# Setup Dev Environment

Install mandatory and optional dependencies on Debian/Ubuntu:

```bash
# Mandatory tools
sudo apt update
sudo apt install build-essential cmake pkg-config

# Optional headers/libs for full features
sudo apt install libcfitsio-dev libpng-dev     libavformat-dev libavcodec-dev libswscale-dev libavutil-dev     libomp-dev
```
