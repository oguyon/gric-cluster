# Feature Planner

This skill guides the agent through scoping, planning, and getting approval for features or changes.

## Feature Design Workflow
1. **Research & Requirements:** Understand the goal, locate affected files, and analyze dependencies.
2. **Draft Plan:** Put together the technical changes in `implementation_plan.md` using the standard template.
3. **Review:** Present the plan to the user for feedback.
4. **Iterate:** Address feedback until the plan is approved.
